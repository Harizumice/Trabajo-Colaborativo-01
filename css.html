COMANDOS EN GIT
Nombre de la etiqueta Para qué se utiliza
font-size: 100px; establece el tamaño de la fuente.
color: red; establece un color en la fuente, para un color personalizado se puede usar el rbg(0,0,0) y cambiar sus valores o la posición de la rueda de colores.
class="NombreClase" Sirven para establecerle un estilo específico a cada elemento de HTML, para llamarlas se utiliza un .seguido del nombre de la clase.
id="NombreID" Son como las clases pero ya no se usan tanto, lo importante al usarlos es no repetir más de un elemento de HTML con el mismo ID. para ser llamados se usa un # seguido del nombre del ID.
background/background-color: blue; Establece el color del cuerpo de la página.
display Nos ayuda a controlar dónde se va a ver un elemento HTML dentro de la pantalla para estructurar nuestra página web. (la más usada es display: flex)
justify-content: Determina dónde se ubicarà nuestro elemento en la página (ya sea izquierda, derecha o el centro).
text-align:" Es como el anterior pero funciona en el texto.
font-family: Establece una fuente para nuestro texto, una manera de conseguirlas es por medio de google fonts, con el segundo link
background-image: Inserta una imagen en el fondo del cuerpo de la página.
